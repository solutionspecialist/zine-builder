# Garford Zine Builder 🎨
An open-source, DIY tool for creating 8-page zines from a single sheet of paper. Developed for the **Garford Arts Fest** (Aug 8, 2026).

## 🚀 Features
- **Modular Design:** Separate image processing and PDF generation.
- **8-1 Spread Support:** Allows for wraparound cover art.
- **Mobile-Friendly:** Designed for use at live festival stations.

## 🛠️ Setup
1. `python -m venv .venv`
2. `source .venv/bin/activate` (or `.venv\Scripts\activate` on Windows)
3. `pip install -r requirements.txt`
4. `streamlit run app.py`
